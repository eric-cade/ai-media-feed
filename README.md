# ai-media-feed
A generative, topic-based media feed built with Godot Engine.

**AI Media Feed (Open Source Prototype)**

A generative, topic-based media feed built with Godot Engine
This project explores a new kind of media ecosystem:

-AI-generated posts

-Topic categories (similar to subreddits)

-Endless vertical feed

-Ranking & interaction system

-Optional backend for local or cloud AI models

-This open-source release includes the Godot UI client, basic feed logic, and a placeholder backend for testing.

**Current Features**

-Scrollable generative feed

-Topic selection system

-Template-based post generation

-Local dummy backend (no API keys required)

-Extensible architecture for contributors
Roadmap

**UI improvements**

-Post evolution system

-Mobile-friendly design

See roadmap.md for details.

**Running the Project**

-Clone the repo

-Open godot/ in Godot 4.x

-Run the project

-Posts will be generated using the dummy backend

For real AI generation, implement your own backend using the structure in backend/api_structure.md.
